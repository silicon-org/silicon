# Examples

Silicon is a modern hardware description language focusing on ergonomic syntax and rich metaprogramming through phased compile-time execution.

This part of the documentation is a collection of annotated programs that illustrate the language's features and idioms.
Each example is a self-contained Silicon program that you can compile with `silc`.
For a more structured introduction to the language, see the {{< page-link "/language" >}} section.

- {{< page-link "hello-world" >}} - Start with the simplest possible Silicon program.
- {{< page-link "basics" >}} - Functions, variables, types, operators, and other fundamentals.
- {{< page-link "phased-execution" >}} - Use `const` arguments to move computation into the compiler and generate specialized code.
