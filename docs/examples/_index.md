# Examples

Silicon is a modern hardware description language focusing on ergonomic syntax and rich metaprogramming through phased compile-time execution.

This part of the documentation is a collection of annotated programs that illustrate the language's features and idioms.
Each example is a self-contained Silicon program that you can compile with `silc`.
For a more structured introduction to the language, see the [Language]({{< relref "language" >}}) section.

- [Hello World]({{< relref "hello-world" >}}) - Start with the simplest possible Silicon program.
- [Basics]({{< relref "basics" >}}) - Functions, variables, types, operators, and other fundamentals.
- [Phased Execution]({{< relref "phased-execution" >}}) - Use `const` arguments to move computation into the compiler and generate specialized code.
